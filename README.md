# Análisis ConnectaTel LATAM

## Objetivo
Analizar el comportamiento de usuarios de telecomunicaciones
en LATAM para identificar segmentos y patrones de uso.

## Datasets utilizados
- plans.csv → información de planes disponibles
- users_latam.csv → datos de usuarios registrados
- usage.csv → registros de uso por usuario

## Etapas del análisis
1. Exploración inicial de datos
2. Detección y limpieza de valores nulos y sentinels
3. Estandarización de fechas
4. Agrupación de uso por usuario
5. Visualización de distribuciones y outliers
6. Segmentación de clientes por uso y edad
7. Insight ejecutivo

## Cómo ejecutar
1. Abre el archivo en Google Colab
2. Ejecuta las celdas en orden
3. Los datasets deben estar en /datasets/
